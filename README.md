# [PROZ] Projeto Petshop

Um portal de petshop simples desenvolvido com **HTML**, **CSS** e **JavaScript**.

Este projeto é uma demonstração de um portal de petshop funcional estilo site institucional.

## 🛠 Tecnologias Utilizadas

- **HTML**: Estrutura da página.
- **CSS**: Estilização da interface.
- **JavaScript**: Lógica das operações e interatividade.

## 🎯 Funcionalidades

- **Formulário simples de Cadastro de Animais (apenas frontend)**
- **Calculadora de Ração (apenas frontend)**

## 🚀 Como Usar

1. Clone ou baixe o repositório:
   ```bash
   git clone https://github.com/otechlead/petshop.git

2. Abra o arquivo index.html no seu navegador.

Você verá a interface da petshop pronta para uso!

## ⚙️ Como Funciona

1. O HTML estrutura os botões da petshop.

2. O CSS cuida da parte visual, tornando a interface responsiva e amigável.

3. O JavaScript captura as entradas do usuário, realiza os cálculos e exibe o resultado no visor da petshop.
